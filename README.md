# Seleniumscripts

A Maven project for Selenium automation testing with TestNG.

## Dependencies Included
- Selenium WebDriver 4.15.0
- TestNG 7.8.0
- WebDriverManager 5.6.2
- Apache Commons Lang3 3.13.0

## Running Tests
```bash
mvn test
```

```
Seleniumscripts/
├── src/

```
